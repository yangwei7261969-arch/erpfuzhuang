# 项目上下文

### 版本技术栈

- **Framework**: Next.js 16 (App Router)
- **Core**: React 19
- **Language**: TypeScript 5
- **UI 组件**: shadcn/ui (基于 Radix UI)
- **Styling**: Tailwind CSS 4
- **数据存储**: localStorage (模拟后端数据库)

## 项目概述

服装生产ERP管理系统，实现从订单到财务的全流程管理，支持扎号追溯、扫码报工、成本核算、外协加工、工艺路线管理、生产看板大屏等全功能。

### 核心功能模块

1. **首页** - 系统概览、待办事项、快捷入口
2. **生产看板** - 实时订单进度、生产线产量、库存预警
3. **订单管理** - 订单录入、色码矩阵、工艺要求
4. **BOM管理** - 物料清单、扎号管理、与订单强联动
5. **工艺路线** - 标准工序库、工艺路线配置
6. **裁床管理** - 裁床单、分扎、领料
7. **车间报工** - 扫码报工、按工艺路线走工序
8. **尾部管理** - 整烫、质检、包装、入库
9. **外协管理** - 印花/绣花/洗水/缝制/整烫外协
10. **财务成本** - 成本核算、应收应付
11. **审核管理** - 订单/BOM/财务审核
12. **库存管理** - 物料入库/出库/盘点
13. **员工班组** - 员工档案、班组配置、计件工资
14. **客商管理** - 客户/供应商档案
15. **报表中心** - 生产统计、成本分析、效率分析
16. **扫码端** - 扎号扫描、工序报工、员工打卡
17. **系统日志** - 操作记录、审计追溯（不可删除）
18. **打印模板** - 自定义纸张、条码、LOGO
19. **用户管理** - 用户权限配置
20. **系统设置** - 外观设置、主题切换

### 主题系统

- 支持**亮色/暗色模式**切换
- 支持**6种主题颜色**自定义：经典黑、科技蓝、自然绿、优雅紫、活力橙、热情红
- 白黑风格设计（浅色：白底黑字，深色：黑底白字）

## 包管理规范

**仅允许使用 pnpm** 作为包管理器，**严禁使用 npm 或 yarn**。

## 开发规范

- **数据持久化**: 使用 localStorage 模拟后端数据库，数据以 `erp_` 前缀存储
- **组件规范**: 使用 'use client' 标记客户端组件
- **类型安全**: TypeScript 严格模式，所有类型定义在 `src/types/` 目录
- **样式规范**: 使用 Tailwind CSS + shadcn/ui，支持主题切换

## 核心业务逻辑

### BOM与订单联动
- 选择订单后自动带出所有基础信息和工艺要求
- BOM明细按颜色×尺码矩阵生成扎号

### 工艺路线管理
- 绑定BOM，车间报工必须按路线走
- 不跳工序，确保生产顺序

### 外协管理
- 支持5种类型：印花、绣花、洗水、缝制、整烫
- 自动生成外协应付单

### 主题系统
- ThemeProvider 管理全局主题状态
- 支持 localStorage 持久化主题配置
- 亮色/暗色模式 + 6种主题颜色
